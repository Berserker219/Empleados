# Empleados
 El proyecto esta desarrollado con Django como backend, en frontend se desarrollo con html,css y javascript puro
Un proyecto donde se puede registrar empleados y departamentos de una empresa, con una analisis sencillo y una funcionalidad de recursos humanos.
